# 🛡️ Cyber Safety Assistant

## 📌 Project Overview

Cyber Safety Assistant is a web-based application designed to help users identify potentially suspicious messages and phishing attempts.

The user can enter a message and analyse its content. The system evaluates the message based on common cyber threat indicators such as urgency language, OTP requests, suspicious links, verification requests, and bank-related content.

A risk percentage is generated and displayed through a color-coded progress bar, helping users quickly understand the level of risk associated with the message.

---

## 🎯 Problem Statement

Many users fall victim to phishing attacks, scam messages, and fraudulent links due to a lack of cybersecurity awareness.

This project aims to improve cyber awareness by providing a simple tool that helps users identify suspicious messages before taking any action.

---

## ✨ Features

* Analyze suspicious messages
* Risk Percentage Calculation
* Color-Coded Risk Indicator

  * Green → Low Risk
  * Yellow → Medium Risk
  * Red → High Risk
* Detailed Reason Analysis
* User-Friendly Interface
* Instant Risk Assessment

## ⚙️ How It Works

1. Enter a message in the input field.
2. Click the **ANALYSE** button.
3. The system calculates the risk percentage.
4. A risk category is displayed:

   * Low Risk
   * Medium Risk
   * High Risk
5. Click the **REASON** button to view the detected indicators.

---

## 🚨 Risk Indicators

The application checks for common phishing indicators such as:

* OTP requests
* Urgency language
* Verification requests
* Bank-related messages
* Suspicious links
* Suspicious actions
